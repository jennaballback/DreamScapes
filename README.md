# DreamScapes
Dream Journal and Interpretation App

DreamScapes is a mobile application that makes dream journaling simple, engaging, and insightful. Users can log their dreams in a private, easy-to-use interface, while also exploring optional interpretations generated by AI and informed by cultural and symbolic perspectives. The app encourages consistency with streaks, badges, and yearly recaps that highlight patterns, recurring symbols, and personal growth over time.
